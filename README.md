# AgentClinicSeri

A simple, interactive counter application built with **TypeScript** and **Vite**, designed as a playground for WebStorm features.

## 🚀 Features

- **Interactive Counter:** Increase or decrease a counter value using buttons (`+1`, `+2`, `-1`, `-2`).
- **Value Constraints:** The counter logic includes bounds-checking, resetting the value if it exceeds `100` or drops below `-100`.
- **Modern Tech Stack:** Utilizes TypeScript for type safety and Vite for a fast development experience.
- **WebStorm Optimized:** The codebase includes integrated tips and shortcuts to help you explore WebStorm's IDE capabilities.

## 🛠️ Tech Stack

- **Framework:** [Vite](https://vitejs.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** Vanilla CSS
- **Formatting:** Prettier

## 🏃 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (latest LTS recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/charoensri/WebStormGeminiAgentClinicSeri.git
   ```
2. Navigate to the project directory:
   ```bash
   cd AgentClinicSeri
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Build

To create a production-ready build:
```bash
npm run build
```
